# Power-BI-Report---Finance-Data
*The PBIX file type requires download for viewing*
# Why?

This Power BI Report builds on my existing knowledge of the tool, spending time transforming and shaping the raw data, cleansing and doing an analysis of the profile of the data and adding explicit measures along with relevant calculated columns to derive more insights.

# The Data

* One excel file was used in this report so the data model was pretty basic and there was no need to establish any table relationships because only one table was leveraged.

* The data itself consisted of financial data based on dummy data for countries, split up by segments.

* My main goal was to show an executive level finance overview of the data and then drill into the product detail to look at specifics. I also threw in a quick map detail with tooltips that show some more detail.
* 
# Power BI WorkFlow Steps Used

* Loading and Transforming data in Power Query Editor

* Cleansing data and Brief Profile of the data

* Calculated columns and explicit measures created

* Visuals such as cards, matrices, graphs and slicers incorporated  in report view
